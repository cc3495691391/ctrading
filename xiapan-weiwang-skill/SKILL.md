# 下盘为王 · 足球投注分析系统

## Quick Start

访问 `assets/index.html` 即可打开完整的投注分析界面。

## 核心功能

### 投注方式
- **单关**：单场比赛
- **串关**：2串1 ~ 8串1
- **系统过关**：3串3 ~ 8串247
- **双选/全包**：同一场比赛可多选结果

### 资金管理
- 初始资金 ¥120,000（可调整）
- 每日限额 5%，每周限额 25%，每月限额 50%
- 利润自动滚入，亏损自动扣减
- 资金耗尽自动禁止投注

### 分析系统
- **打分层**：5个因子评分（基本面支撑、初盘位置、冷门、热度、时间窗口）
- **状态系数**：根据最近战绩自动调整（连胜衰减、连亏止损）
- **执行力系数**：根据历史胜率、近5场、稳定性计算
- **仓位建议**：根据总分 + 系数自动计算

### 自动更新
- 每天 5:00 BJT 通过 cron 自动更新赛程数据
- 支持的联赛：英超、西甲、德甲、意甲、法甲、欧冠、欧联杯、世界杯
- 时间规则：0:00-4:59 BJT 的比赛归前一日

## Assets

- `assets/index.html` - 主界面（含完整 HTML/CSS/JS）
- `assets/template.html` - 模板文件（用于每日更新）
- `assets/generate.py` - 每日赛程生成脚本（调用 sports-skills CLI）

## Cron

已预置 cron job（id: `72156f48-bc63-43be-bbe1-134a3ad51269`），每天 5:00 BJT 自动执行 `generate.py` 更新赛程。

## Using from Other Skills

引用本技能时的触发词：投注、赔率、五大联赛、下盘、串关、系统过关、竞彩分析。
